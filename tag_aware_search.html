Tag aware search

Attempt to create a crude interface for a tag aware search mechanism modeled on the Orlando Googlesque search. This interface is meant to start a discussion.

Note: reloading the page after a repository change should show the updated content within 1-2 sec.

To see hits, search for "zzzMARKzzz"

Note: the facets on the right-hand side have a 'document' count (i.e. number of Fedora Objects that match the query and contain the specificed element). Elements are not removed from the list until the page is reloaded (i.e. there is no clear/reset as I'm not sure how to impliment at this moment).

The 'apos' and '{}' are required at this stage of development.
